3. Known Issues

To report issues or provide feedback about this instrument driver, please send an email to instrument.drivers@ni.com.
  1. Not all VIs are supported by all instrument models. This is documented throughout the instrument driver. Refer to instrument documentation for more details pertaining to your specific model.
  2. The TDS 5000 series instruments do not support histogram measurements, despite being listed in the manual.
  3. For the older TDS models, models intermittently failed to process commands when multiple commands were issued. To address this issue delays were added after the VISA writes. This might not be an issue with the newer models.
  4. Tektronix sometimes may update firmware which might not be backwards compatible. If you can not use Configure Trigger (Window).vi and Configure Trigger (Video).vi, please update the firmware to the latest version.
  5. Although all trigger VIs are implemented in the driver, not all of them have been tested with the live instrument. The VIs that haven't been tested are the ones that relative with VIDeo, I2C, CAN, SPI, RS232 trigger type.
